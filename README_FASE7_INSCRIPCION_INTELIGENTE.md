# Fase 7 - Inscripción inteligente

Cambios principales:
- Panchito puede iniciar una inscripción desde una frase natural.
- Si ya detectó deporte, edad y categoría recomendada, usa esos datos.
- Flujo completo por chat:
  1. Nombre y apellido
  2. Edad / fecha de nacimiento
  3. DNI
  4. Si es socio/a
  5. Teléfono
  6. Email
  7. Observaciones
  8. Confirmación final
- Guarda la inscripción en la pestaña Inscripciones con estado Pendiente.
- Genera resumen final y link para enviar la solicitud por WhatsApp a Administración.
- Permite modificar nombre, edad o teléfono antes de confirmar.
- Se puede cancelar escribiendo MENÚ.
